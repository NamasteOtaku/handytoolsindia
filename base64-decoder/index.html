About Base64 Decoding

How It Works

Base64 decoding converts ASCII text back into its original binary format. This tool automatically detects if the decoded data is text or an image.

Usage Tips

  • Supports text, PNG, JPG, GIF and WebP
  • Decoded images can be downloaded
  • Works with all modern browsers
